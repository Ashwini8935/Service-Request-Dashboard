# Service-Request-Dashboard
Service Request &amp; SLA Dashboard using PostgreSQL and Power BI

## Project Overview
Built a service analytics dashboard to monitor tickets, SLA breaches, engineer workload, issue trends, and customer locations.

## Tools Used
- PostgreSQL
- SQL
- Power BI

## Key Features
- Open vs Closed Tickets
- SLA Breach Tracking
- Engineer Performance
- Monthly Ticket Trend
- Issue Type Analysis
- Location-wise Requests

## Business Impact
Helps improve service response time, workload planning, and customer support visibility.
